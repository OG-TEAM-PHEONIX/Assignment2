Tools

To build the structure and foundation of the groups' website, the HTML5 markup language was used in unison with the free, open-source CSS framework, Bootstrap. Bootstrap was chosen for it's ease of use and time-saving capabilites in terms of coding, as well it's in-built, responsiveness capacities. Custom CSS was used to overwrite certain Bootsrap aspects and for further website customization. This was all achieved in the preferred IDE of Visual Studio Code. These files are hosted on the Team Phoenix GitHub Repository and available through the GitHub Pages.
